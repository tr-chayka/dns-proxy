1. Сборка:

cmake .
make

2. Настройки и blacklist

Настройки хранятся в файле settings.txt, при запуске приложение выводит в stdout считанные настройки.
В файле blacklist.txt хранится список фильтруемых доменных имен. 
Для простоты фильтруются имена только по полному совпадению.
Например, если в blacklist.txt указано имя www.google.com, то фильтроваться будет только оно, а например google.com - не будет.

3. Тестирование

Автоматических тестов нет, тестирование ручное утилитой dig
dig @localhost -p <port> <name to resolve>
